# Space Espionage
Space Espionage is an interactive story-based game created for the final of a creative code class. It is written primarily in Javascript with the p5.js library for creative code. Additionally, it utilizies computer vision, specifically the Handpose library by Tensorflow and p5.speech to allow for user input that goes beyond buttons and a controller.
